# 🛒 Amazon UI Clone

A front-end clone of the Amazon homepage built using pure HTML and CSS.  
This project focuses on layout structuring, responsive design, and clean styling without using any JavaScript or frameworks.

---

## 🚀 Tech Stack

- HTML5  
- CSS3  

---

## ✨ Features

- Fully structured homepage layout  
- Navigation bar with search section  
- Hero banner section  
- Product grid layout  
- Footer section  
- Responsive design using Flexbox  

---

## 🎯 Purpose

This project was built to strengthen core front-end fundamentals, improve layout skills, and practice real-world UI replication using only HTML and CSS.

---

## 📂 How to Run

1. Clone the repository  
2. Open `index.html` in your browser
